# **App Name**: Dollar Leap

## Core Features:

- Headline Generator: Generates persuasive headlines for the sales page using AI, focusing on benefits and urgency. The AI will tool determine the correct emphasis.
- Problem Identifier: Clearly outlines the current problems faced by the target audience (low salaries, limited opportunities, etc.).
- Benefit Highlighting: Showcases the benefits of the course, such as earning in dollars and achieving financial freedom.
- Testimonial Section: Displays testimonials and success stories from course participants.
- FAQ Section: Answers frequently asked questions to address potential concerns.
- Call to Action Buttons: Prominent CTAs placed strategically throughout the page, encouraging immediate action.
- Urgency Indicators: Visual elements indicating scarcity, like limited spots or expiring promotions.

## Style Guidelines:

- Background: Red with a subtle gradient (#FF4136) to evoke urgency and passion.
- Primary: White (#FFFFFF) for clear readability of text content.
- Accent: Blue (#0074D9) for call-to-action buttons and highlighted elements, symbolizing trust and professionalism.
- Headlines: 'Poppins' (sans-serif) in Black weight for impactful titles and calls to action. Note: currently only Google Fonts are supported.
- Subtitles and descriptions: 'Poppins' (sans-serif) in Light weight for easy-to-read supporting text. Note: currently only Google Fonts are supported.
- Use modern, clean icons related to financial freedom and global opportunities.
- Subtle pulsing animation on call-to-action buttons to draw attention.
